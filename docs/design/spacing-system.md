# Spacing System

## Philosophy

Whitespace is part of the design.

Generous spacing creates a premium experience.

Never crowd content.

---

## Layout

Maintain consistent spacing throughout the application.

Sections should have generous vertical spacing.

Containers should provide comfortable horizontal padding.

---

## Components

Every component should have breathing room.

Avoid stacking elements too closely.

Related items should appear visually grouped.

---

## Alignment

Use consistent alignment across all sections.

Avoid random spacing values.

Maintain visual rhythm.

---

## Responsive Design

Spacing should adapt naturally across screen sizes.

Mobile should feel spacious without wasting space.

Desktop should emphasize clarity.

---

## Golden Rule

When uncertain, prefer slightly more whitespace over cramped layouts.
